## 🚀 Core Value Proposition

This is not just a portfolio; it's a technical demonstration of **Architectural Precision** and **Design Excellence**.

### 🛠️ Technical Stack
- **Core Framework**: Next.js 15 (App Router, Server Components)

- **Export Engine**: html-to-image (High-Fidelity UHD Asset Generation)
- **Data Handling**: React Hook Form + Zod (Strict Schema Validation)
- **Messaging**: EmailJS (Asynchronous Communication Pipeline)

## ✨ Key Technical Features

### 1. **Context-Aware AI Assistant**
Built on Google Genkit, the integrated AI chatbot provides first-class support for inquiries regarding technical projects, experience, and contact details. It leverages a localized knowledge base to provide instant, human-grade technical responses.

### 2. **UHD Digital Identity Card**
A high-fidelity virtual business card with a built-in UHD PNG export engine.
- **High-Res Rendering**: Utilizes `pixelRatio: 3` for sharp, competition-ready social sharing.
- **Glassmorphic Logic**: Real-time canvas capture that preserves complex CSS attributes like `bg-clip-text` and nested gradients.

### 3. **Themed Strategic Collaboration Interface**
A production-grade inquiry system featuring:
- **Honeypot Security**: Automated spam protection.
- **Schema Validation**: Real-time field verification via Zod.
- **Theme Awareness**: Engineered for perfect accessibility in both light and dark modes.

### 4. **Micro-Interaction System**
Optimized layout logic that uses Framer Motion variants for orchestrated "Reveal-on-Scroll" sequences, ensuring a premium "Elite" first impression.

## 🛠️ Installation & Development

### 1. Clone & Install
```bash
git clone https://github.com/Thanuka9/Portfolio.git
cd Portfolio
npm install
```

### 2. Environment Setup
Create a `.env.local` file with the following:
```env
GOOGLE_GENKIT_API_KEY=your_key
NEXT_PUBLIC_SERVICE_ID=your_id
NEXT_PUBLIC_TEMPLATE_ID=your_id
NEXT_PUBLIC_KEY=your_key
```

### 3. Launch Development
```bash
npm run dev
```
Navigate to `http://localhost:3000` to view the high-end experience.
