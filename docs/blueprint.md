# **App Name**: Ellepola Portfolio

## Core Features:

- Hero Section: Display professional headshot, name, title, and a brief tagline on the hero section.
- Skills & Experience: Showcase key skills, education, and experience in a clear, concise format.
- Project Showcase: Implement a portfolio section to highlight projects, with embedded PDF view for thesis, plus links to GitHub and live demos.
- CV Download: Enable downloading of the CV PDF.
- Contact & Links: Include contact information and links to GitHub and LinkedIn profiles.
- SEO Optimization Tool: Generate SEO meta tags for better search engine visibility based on the provided keywords.
- Testimonials Section: Display testimonials or references to build credibility and trust, if available.

## Style Guidelines:

- Primary color: Slate blue (#708090) for a professional and trustworthy feel, derived from the concept of blending technical expertise with analytical insights.
- Background color: Light gray (#F5F5F5) for a clean and modern look, providing good contrast with the content.
- Accent color: Teal (#008080) to highlight key information and call-to-action buttons, chosen for its association with clarity and innovation.
- Body and headline font: 'Inter', a grotesque-style sans-serif, providing a modern, neutral look.
- Use a consistent set of minimalist icons to represent skills and competencies.
- Employ a grid-based layout for project showcases and skills sections to maintain order and readability.
- Incorporate subtle transition effects when navigating between sections to enhance user experience.
